# Examples

Framework and domain examples for quick onboarding:

- `django_shop/` - Django ecommerce checkout discount
- `drf_orders/` - DRF order risk gate
- `fintech_kyc/` - fintech KYC transfer limit
- `fastapi_limits/` - FastAPI transfer limit
- `flask_checkout/` - Flask checkout eligibility
- `celery_fraud/` - Celery fraud queue routing
- `plain_python/` - framework-agnostic feature gate

Each example includes:

- rule JSON (`rules/*.json`)
- decorator-annotated Python service function
- `logiclock graph` command in local README
